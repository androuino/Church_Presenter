# How to install this app
1. Copy and paste this folder to any directory you want.
2. Open this folder and install the Java.exe
3. Follow the instruction of the Java installation
4. After Java installation, you can now use the system.
5. To start the system, just double-click the run.bat file. 
   1. Please take note that there are two run files in this folder.
   2. Make sure that you are clicking the windows batch file.
   3. To view the system, open up a browser and type-in
