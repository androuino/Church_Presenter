# Church Presenter App
Church Presenter v2.0
This app still in development and you may experience unexpected behavior. In that case, please create an issue.

I also apologize for lacking of documentation on how to use this app as I am very busy with my sched.

The purpose why I created this app is because my original plan is to run this in Raspberry pi and control it remotely as long as both devices (Rpi and a computer) are connected to the same network.
If you don't have a raspberry pi, you can still use this app using your normal computer.

## The download links
[windows/linux/mac](https://drive.google.com/drive/folders/127Ya2byB306WCGRkNZA4TeRYnq7IK5mM?usp=sharing)

## To use in Windows
        1. Unzip or just copy the downloaded folder to your preferred location
        2. Download the javajdk or javaSE 11 and install it
        3. To run the app, double click the run.bat. This will open up a terminal, don't close it as it will terminate the app
        4. It is recommended to use the google chrome browser if installed, if not you can use the default browser of windows (edge)
        5. Type in http://localhost:5555/admin
        6. To login, just type "admin" as username and password

## To use in Linux/Mac
        1. Unzip or just copy the downloaded file to your preferred location
        2. Download the java openjdk 11 and install it
        3. To run the app. execute the ./run located in the app's folder
        4. I recommended to install the google chrome browser for better experience
        5. Once running, open up a browser and type in http://localhost:5555/admin
        6. The login is "admin" for username and password

## To compile this project
       1. I highly suggest to use the IntelliJ IDE
       2. in terminal, execute this command -> npm install
       3. in Linux, just run the compile file, ./compile
       4. once finished, this will create the deploy folder and all the files needed to run this app
